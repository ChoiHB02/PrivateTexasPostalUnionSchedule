## 게임 컨셉
- 더 타워 : 타워 디펜스를 서부 판타지로 개조
- 중앙 기지 -> 이동하는 마차
- 외곽 오브 4개 -> 동료 4명 (인디언 루, 카우보이 잭, 앤더슨 부인, 대장장이 칼)
- 몬스터 -> 동물, 인디언, 탈영병, 강도단등 
- 보스 
    - 웨이브 10의 배수마다 보스가 나온다.
    - 원작은 hp가 높은 정도
    - 우리도 비슷한 상황인데 이를 중간보스라 정리
    - 100의 배수마다 보스가 나온다.
    - 외형은 해당 유닛을 크기를 기워서 제작
    - 액션 패턴은 보스마다 다름 
    
## 버전 정의 
- 동작 샘플 : 기본 전투가 이루어지면 공격력, 공격등 스탯 업그레이드 작동
- 알파 : 게임 종료
- 베타 : 그래픽등 종료
- 마스터 : 아웃 게임 결정
